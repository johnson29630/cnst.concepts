Footnotes

  1. For low consumption coils (LC1D09–D38 only), only one front-mounted two-contact block allowed. No side-mounted contact blocks allowed.
  2. For spring terminal versions of these blocks, add a 3 to the end of the catalog number (for example, LADN223). For slip-on versions, add 9 to the end of the catalog number (for example, LADN229).
  3. Including 1 N.O. + 1 N.C. make-before break overlapping contacts.
  4. This block cannot be added to the LC1D 09–D38 contactors; a maximum of 2 blocks can be mounted on the LC1D40A-LC1/LP1D80 contactors only.
  5. 1 block may be added to the left side of LC1D09–D38, AC coils only; only 1 block may be added to either side of the LC1D40A-D80 contactors, AC coils only. Cannot be installed on TeSys D contactors with DC coils.
  6. Device supplied with 4 ground terminal points.
  7. For spring terminal versions of these blocks, add a 3 to the end of the catalog number (for example, LADT23).
  8. Scale range is expanded between 0.1 and 0.6 seconds on the dial for more accurate settings at the lower end of the range.
  9. Switching time between the opening of the N.C. contact and the closing of the N.O. contact: 40 ms ± 15 ms .
  10. To complete the catalog number, add the coil voltage code from Coil Voltage Codes for LA6DK Mechanical Latch Blocks (for example, LAD6K10F).
  11. Does not include internal coil clearing contact.
  12. Low consumption DC contactors (and relays) (code coil ●L) are not compatible with the LAD6K10● mechanical latching blocks.
