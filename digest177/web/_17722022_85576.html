Type G—Pumptrol Pressure Switch

Class 9013 / Refer to Catalog 9013CT9701

Type G Pressure Switch Selection and Features

Shown with Form X

File E12158
CCN NKPZ

File E12443
CCN NOWT
Haz Loc

File 25490

File 26817

Haz. Loc.

Class 9013 Type G Pumptrol pressure switches are designed to control electrically driven water pumps and air compressors. These devices cover higher electrical ratings for directly controlling motors in pump and compressor applications.

  • Contacts open on pressure rise.

  • Diaphragm actuated.

  • For electrical ratings, see FHG Pressure Switch Selection and Features.
    For repair parts kits, see Renewal Parts for Class 9012–9038 Devices.

Ordering Information

  • Specify Class 9013 Type G.

  • Select the pressure code from Pressure Codes, and add the code to the end of the Type number. Ensure that the pressure rating of the code falls within the limits of the device. See Selection Tables.

  • To order special features, add the appropriate Form letter to the Class and Type. Arrange Form letters in alphabetical order when ordering more than one special feature.
