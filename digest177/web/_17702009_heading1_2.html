Selection Information

Class 4141 / Refer to Catalog 4100CT0701

Selection Information

  • Review local utility requirements to ensure that metering equipment meets their standards.

  • Check local utility to determine available fault current at the meter center.

  • Using the SCCR table:

    • Select meter center configuration, main lugs only (Six Disconnect Rule), or remote main, main circuit breaker, or main fusible switch.

    • Read down to select SCCR equal to, or greater than desired rating.

    • Read across to select branch unit tenant circuit breaker type.

    • Continue reading across to select EZM main device type.
