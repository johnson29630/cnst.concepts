Contactors, Type S

Class 8502 / Refer to Catalog 8502CT9701

General Information

Type SCO2

Size 1, 3-Pole Contactor

Class 8502 Type S magnetic contactors are used to switch heating loads, capacitors, transformers, and electric motors where overload protection is provided separately. Class 8502 contactors are available in NEMA Sizes 00–7. Type S contactors are designed for operation up to 600 Vac, 50–60 Hz.

Auxiliary Units

Auxiliary contacts and power poles can be added in the factory or the field on all Type S starters and contactors. Auxiliary Units—Class 8502 and 8536 shows the maximum number of auxiliary units, in addition to the holding circuit contact, that can be added to a given size starter or contactor. In addition, it is possible to add a second internal contact on NEMA Size 0, 1, and 2 contactors and starters.
