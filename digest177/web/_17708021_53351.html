Dual Cable Operating Mechanisms for Square Dâ„¢ Circuit Breakers

9422CSFD33

Dual Cable Operating Mechanisms are designed for use with Square D brand PowerPactâ„¢ D, H, J, and L circuit breakers through 600 A frame sizes. The cable mechanisms allow for a single handle operator, Class 9422Ax, to operate both circuit breakers. The cable mechanism is designed especially for tall, deep enclosures where placement flexibility is required. There are numerous cable arrangements to choose from to accommodate many applications.

Features

  • Separate cables for each circuit breaker

  • Rugged metal flange handle operator

  • Maximized flexibility of circuit breaker placement for existing and new applications

  • Control panel can be fed from two separate supply voltages (if required)

  • Dual mechanism allows both separate supply voltages to be controlled by a single handle to improve security features
