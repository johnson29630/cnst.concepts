Footnotes

  1. When ordering, add prefix 9001 to the catalog number.
  2. The universal push button operators contain one each of the following color inserts: black, red, green, yellow, orange, blue and white.
  3. ■ Choose color code from Color Codes for Type K Operators.
  4. Knob has the words “Emergency Stop” in raised letters highlighted in white for readability.
  5. For 1-3/8 in. or 2-1/4 in. Dia. Knob:
    a) Order Type -20 or -21 knob from Additional Accessories for Type K and SK Operators.
    b) Order 9001K54 adapter—allows Type -20 or -21 knob to fit on push pull operators. Voids UL and NEMA 6 rating.
    c) Can order assembled operator by adding color code to Type -20 or -21. Example: 9001KR9R would be 9001KR9R20 or 9001KR9R21.
  6. These operators can be ordered complete with contact blocks. For maximum block usage, see “H” Codes. Add the chosen “H” number to the end of the operator.
  7. For contact sequences, see Contact Sequences
  8. ▼ Choose one color from the Color Codes table here, and insert the color code in Type number. Example: KR9 with a yellow knob = KR9Y
  9. For color codes, see Color Codes for 30 mm Multifunction Operators.
  10. To obtain a red knob with “Push Emergency Stop” printed on the red knob—substitute “R05” in place of “R”
  11. ▲ Add the voltage assembly code as chosen from Standard Light Modules for Types K, SK, and KX Control Units. Example: KR8P with a 277 V 50–60 Hz voltage = KR8P8.
  12. The knob must be the same color as the LED light module chosen, for example, for a green LED, use a green knob.
  13. On neon light modules, use clear knobs only.
  14. These colors are not available on illuminated push-pull operators.
  15. These operators can be ordered complete with contact blocks. For maximum block usage, see “H” Codes. Add the “H” number to the end of the operator type number.
  16. Add the color code as chosen from the color cap table. Example: K2L25 with a blue 1–3/8 in. mushroom button = K2L25L20.
  17. ■ Add the voltage assembly code as chosen from Standard and Shallow Depth Light Modules. Example: K2L with 240 Vac/Vdc = K2L25.
  18. On neon light modules, use clear color caps only.
  19. The cap must be the same color as the LED light module chosen, e.g., for red LED, use red color cap.
  20. The only difference between a no guard (K2L_) operator and mushroom button operator is the color cap.
